== Description ==

IN4X Gateway provides facilities for smooth, safe and instant cryptocurrency payments with automated exchange to
 EUR/USD or any other fiat currency at direct exchange rates. Payment processing fees are reasonable and calculated
  depending on business risk levels.
Using our payment system, your customers can easily and instantly pay for the purchases on the website you’ve
 integrated with IN4X Gateway. They can make purchases on any device and without any fees for IN4X-inspired payments.
A branded crypto wallet – IN4X Wallet facilitates safe, easy and instant crypto/fiat payments and exchanges. Level-up
 your business and be a part of a modern world with IN4X.

== Installation ==

= Minimum Requirements =

* WordPress 4.6 or greater
* PHP version 5.6 or greater

= Automatic installation =

1. Go to administrative panel of your shop.
2. In the menu on the left choose the tab “Plugins”, then press “Add a New Plugin”.
3. Type into the search bar “IN4X Crypto Payment” - the plugin will appear at the top of the results list.
4. Press the button “Install”.
5. After the end of installation press the button “Activate”.
6. On the tab WooCommerce go to “Settings”  -> “Payments”, in the list of available payment methods choose
 “IN4X Crypto Payment” and go to payment method editing page “IN4X Crypto Payment”.
7. Copy the field value “URL for Server Response”.
8. After registering on the site https://www.in4xglobal.com/ you receive Public API Key and Secret API Key which are necessary for
 the plugin work. These keys are stored on your personal page in the section “Account” (on the site https://www.in4xglobal.com/app/merchant-e-commerce-integration) on
  the tab “API Keys”. On this tab in the field “Your URL” it is also necessary to insert the field value “URL for the
   Server Response” which you have copied on the previous step.
9. Copy Public API Key and Secret API Key into the corresponding fields in the module settings “IN4X Crypto Payment” -> Save

= Manual installation =

1. The manual installation method involves downloading our plugin and uploading it to your webserver via your favorite FTP
 application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
2. After the end of installation press the button “Activate”.
3. On the tab WooCommerce go to “Settings”  -> “Payments”, in the list of available payment methods choose
 “IN4X Crypto Payment” and go to payment method editing page “IN4X Crypto Payment”.
4. Copy the field value “URL for Server Response”.
5. After registering on the site https://www.in4xglobal.com/ you receive Public API Key and Secret API Key which are necessary for
 the plugin work. These keys are stored on your personal page in the section “Account” (on the site https://www.in4xglobal.com/) on
  the tab “API Keys”. On this tab in the field “Your URL” it is also necessary to insert the field value “URL for the
   Server Response” which you have copied on the previous step.
6. Copy Public API Key and Secret API Key into the corresponding fields in the module settings “IN4X Crypto Payment” -> Save
